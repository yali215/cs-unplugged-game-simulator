## 🎲 三大核心教學模組介紹

本模擬器完美復刻了我們企劃書中的實體不插電教學活動：

1. **二進位翻牌遊戲** 透過卡牌的翻轉與對照，讓小朋友無痛理解電腦底層最基礎的 $0$ 與 $1$ 編碼邏輯。

2. **天秤排序算法** 藉由幫不同重量的物品進行兩兩比較與排序，讓學童親手操作計算機科學中最經典的排序演算法。

3. **大地遊戲控制結構** 將程式設計中的 `if/else` 分支判斷與循環邏輯（Loop），融入好玩的大地遊戲實體指令中，培養空間決策能力。

# 🎮 CS Unplugged Game Simulator
### Logic & Spatial Decision Game Box: An Interactive CS Initiation for Elementary School Students

This project is specially designed for elementary school students in rural areas (such as Dapo, Baoshan, and Chaoyin Elementary Schools). Our goal is to spark students' interest in Computer Science (CS) through interactive, hands-on games—completely free from screen-time vision strain and device limitations!

## 🎲 Core Educational Modules

The simulator perfectly replicates the physical CS Unplugged activities outlined in our project proposal:

1. **Binary Card Game** Through flipping and matching cards, children intuitively understand the fundamental $0$ and $1$ coding logic at the lowest layer of computers.

2. **Balance Scale Sorting Algorithm** By performing pair-wise weight comparisons and sorting objects, students get hands-on experience with the most classic sorting algorithms in computer science.

3. **Real-Life Control Structure** Integrating programming concepts like `if/else` conditional branches and loops into a fun, spatial decision-making game, cultivating computational thinking.

---

## 🛠️ Key Features & Tech Stack

* **Pure Frontend Architecture**: 100% built with HTML5, CSS3, and native JavaScript. 
* **Zero Dependencies & Zero Tokens**: The entire codebase is encapsulated within a single `index.html` file, ensuring lightning-fast loading and offline execution capabilities.
* **Designed for Education**: Colorful, intuitive, and highly accessible user interface tailored for young learners.
